# SearchEngine
A simple Chinese search-engine based on Java.

**Using tools:**
> Eclipse Java EE IDE for Web Developers 4.5.0 
> heritrix-1.14.4 
> lucene-2.4.0 
> htmlparser1_6 
> je-analysis-1.5.1 
> apache-tomcat-7.0.64
